***********************************
   Instalação do Italo Xadrez
***********************************

Para instalar o SDL no ubuntu, execute os seguintes comandos:

sudo apt -y install libsdl2-2.0-0
sudo apt -y install libsdl2-mixer-2.0-0
sudo apt -y install libsdl2-image-2.0-0
sudo apt -y install libsdl2-ttf-2.0-0

Após isto, pode navegar pelo terminal até a raiz do Italo-Xadrez e executar o 
seguite comando: 

./italo-xadrez
